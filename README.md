# Emotion Detector Application

## 📌 Project Overview
The Emotion Detector Application is an AI-based web application that analyzes text input and detects the underlying emotions using IBM Watson NLP services.

The application identifies the following emotions:
- Anger
- Disgust
- Fear
- Joy
- Sadness

It also determines the **dominant emotion** based on the highest confidence score.

---

## 🚀 Features
- Analyze user input text for emotions
- Displays emotion scores for five emotions
- Identifies dominant emotion
- Web-based interface using Flask
- Error handling for invalid inputs
- Unit tested for reliability

---

## 🛠️ Technologies Used
- Python
- Flask
- IBM Watson NLP API
- Requests Library
- Unittest (for testing)
- Pylint (for static code analysis)
